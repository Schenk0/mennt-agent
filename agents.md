# Mennt

Read /BRAND.md when ever you need to understand the brand and the product.

## Repository Layout

- `app/` contains all of the client side code and some serverless APIs.
- `components/` defines the UI components
- `lib/` contains the utility functions and types.
- `mennt-agent/` handles the agentic loop by injecting system prompts, defining tools.
- `convex/` contains the database that stores chats and user metadata. (Yet to be implemented)
- `template/` contains the template that we use to start all Mennt projects.
